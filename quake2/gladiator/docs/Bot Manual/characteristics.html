About: This document contains editing information for the Gladiator bot.
The Gladiator bot is an artificial player for Quake2.
Author: Mr Elusive (MrElusive@demigod.demon.nl)
Gladiator page: http://www.botepidemic.com/gladiator
Last update: 1999-05-05


The Gladiator bot uses over 50 characteristics. Every bot character has it's own set of characteristics. These characteristics are stored in files in the 'bots' folder in the .pak file included with the bot. Every characteristic is indexed using the index values defined in the file chars.h in the .pak file.
Have a look at the chars.h file which the bot currently uses.
